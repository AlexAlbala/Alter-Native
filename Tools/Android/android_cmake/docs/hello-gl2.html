Hello GL2


Introduction
-------------


This example is taken directly from the android ndk samples, except here it builds with
cmake.

  
I. Build the native code
----------------


  cd hello-gl2
  mkdir build
  cd build
  android-cmake ..
  make

  
II. Build the Android App (command line way)
----------------


  cd hello-gl2
  ./project_create.sh
  ant compile
  ant install
